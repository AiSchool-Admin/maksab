# مكسب — نظام إدارة العمل الشامل (Maksab Operations Management System - MOMS)
## الإصدار: 1.0 | مارس 2026
## وثيقة تنفيذية لـ Claude Code
## يتكامل مع: كل ملفات المواصفات السابقة

---

## 🎯 الرؤية

> **نظام عمل واحد يدير شركة كاملة — بشر و AI يعملون معاً بتنسيق مثالي.**
> كل موظف (بشري أو آلي) يعرف بالظبط: ماذا يفعل → متى → كيف → النتيجة المتوقعة.
> Dashboard واحد — كل الأقسام — كل المهام — كل الأرقام.

---

```
═══ دليل العمل: مشرف خدمة العملاء ═══
الوقت المطلوب: 2-3 ساعات يومياً

📋 المهام اليومية (بالترتيب):

1️⃣ الصباح — مراجعة التصعيدات (15 دقيقة)
   → افتح /admin/cs/conversations
   → فلتر: "⚠️ تصعيدات"
   → لكل تصعيد: اقرأ المحادثة → رد يدوياً أو حوّل AI يكمل
   → الهدف: صفر تصعيدات عمرها أكتر من ساعتين

2️⃣ إرسال الرسائل الصباحية (15 دقيقة)
   → افتح /admin/sales/outreach
   → اتبع ترتيب الأولوية (الحيتان أولاً)
   → [📱 واتساب] → الرسالة جاهزة → إرسال → [✅ تم]
   → الهدف: 30-50 رسالة/يوم

3️⃣ الظهر — مراجعة ردود AI (30 دقيقة)
   → فلتر: "🤖 AI فقط" + "آخر ساعتين"
   → تأكد إن ردود سارة مناسبة
   → لو رد غلط: صححه يدوياً + أبلغ عن الخطأ
   → الهدف: مراجعة 20 محادثة على الأقل

4️⃣ المساء — تقرير يومي (10 دقائق)
   → افتح /admin/cs/reports
   → راجع: نسبة الحل | وقت الاستجابة | رضا العملاء
   → سجّل أي ملاحظات أو مشاكل متكررة

📊 KPIs (مقاييس الأداء):
  → وقت الاستجابة: < 30 دقيقة
  → نسبة الحل من أول تواصل: > 80%
  → نسبة AI التلقائي: > 85%
  → تصعيدات يومية: < 10
  → رضا العملاء: > 4.5/5
```
