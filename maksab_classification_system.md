## 🎯 الهدف

> كل عميل في مكسب يجب أن يُصنّف تلقائياً في **شريحة واضحة** تحدد:
> - **كيف نتعامل معه** (رسالة مخصصة)
> - **كم نستثمر فيه** (وقت + جهد + عروض)
> - **ماذا نعرض عليه** (باقة + مميزات)
> - **متى نتصل به** (أولوية التواصل)
> - **من يتولاه** (AI أو بشري أو VIP)

## 📊 نظام التصنيف: 7 شرائح

```
═══ الشريحة 1: 🐋 حوت (Whale) ═══
الشروط: whale_score >= 60
  = تاجر + موثق + إعلانات مميزة + 10+ إعلانات
التقدير: ~2% من المعلنين

الإجراء:
  الرسالة: قالب VIP (acq_welcome_whale_v1)
  الأولوية: 🔴 فوري — أول 5 دقائق
  المسؤول: ممدوح شخصياً (أو أعلى مشرف)
  العرض: باقة Diamond مجاناً أول شهر + مدير حساب مخصص
  المتابعة: يومية
  القيمة المتوقعة: 999+ ج.م/شهر

═══ الشريحة 2: 🏪 تاجر مميز (Premium Merchant) ═══
الشروط: is_business = true AND (is_verified OR has_featured) AND listings >= 5
التقدير: ~8% من المعلنين

الإجراء:
  الرسالة: قالب تاجر مميز (حسب الفئة)
  الأولوية: 🟠 عالية — خلال ساعة
  المسؤول: AI "سارة" + مشرف CS يراجع
  العرض: باقة Gold أول شهر بخصم 50%
  المتابعة: كل 48 ساعة
  القيمة المتوقعة: 499 ج.م/شهر

═══ الشريحة 3: 🏬 تاجر عادي (Regular Merchant) ═══
الشروط: is_business = true AND listings >= 2
التقدير: ~15% من المعلنين

الإجراء:
  الرسالة: قالب تاجر عام
  الأولوية: 🟡 متوسطة — خلال 6 ساعات
  المسؤول: AI "سارة" تلقائياً
  العرض: باقة Silver أول شهر مجاني
  المتابعة: 24/48/72 ساعة (3 رسائل)
  القيمة المتوقعة: 199 ج.م/شهر

═══ الشريحة 4: ✅ بائع موثق (Verified Seller) ═══
الشروط: is_verified = true AND is_business = false AND listings >= 2
التقدير: ~10% من المعلنين

الإجراء:
  الرسالة: قالب بائع موثق
  الأولوية: 🟡 متوسطة
  المسؤول: AI "سارة"
  العرض: "جرّب مكسب مجاناً + ننقل إعلاناتك"
  المتابعة: 24/48/72 ساعة
  القيمة المتوقعة: 15-25 ج.م (boost + featured)

═══ الشريحة 5: 👤 بائع نشط (Active Seller) ═══
الشروط: is_business = false AND is_verified = false AND listings >= 2
التقدير: ~25% من المعلنين

الإجراء:
  الرسالة: قالب عام
  الأولوية: 🟢 عادية — خلال 24 ساعة
  المسؤول: AI "سارة" فقط
  العرض: "انشر مجاناً على مكسب"
  المتابعة: 24/48 ساعة (رسالتين فقط)
  القيمة المتوقعة: 10-15 ج.م (boost)

═══ الشريحة 6: 👤 بائع جديد (New Seller) ═══
الشروط: listings = 1 AND phone IS NOT NULL
التقدير: ~30% من المعلنين

الإجراء:
  الرسالة: قالب ترحيب بسيط
  الأولوية: 🟢 عادية
  المسؤول: AI فقط
  العرض: "جرّب مكسب — أول إعلان مجاني"
  المتابعة: رسالة واحدة فقط
  القيمة المتوقعة: 0-10 ج.م

═══ الشريحة 7: 👻 بدون رقم (No Phone) ═══
الشروط: phone IS NULL
التقدير: ~10% من المعلنين

الإجراء:
  لا تواصل مباشر — ينتظر حتى يُستخرج رقمه
  الأولوية: ⚪ منخفضة
  العرض: لا شيء حالياً
  القيمة المتوقعة: 0
```

  -- 'regular_merchant'  — 🏬 تاجر عادي
  -- 'verified_seller'   — ✅ بائع موثق
  -- 'active_seller'     — 👤 بائع نشط

### دالة التصنيف التلقائي

```javascript
function classifySeller(seller) {
  // الشريحة 7: بدون رقم
  if (!seller.phone) {
    return { tier: 'no_phone', value: 0, priority: 0, emoji: '👻' };
  }
  
  // الشريحة 1: حوت
  if (seller.whale_score >= 60) {
    return { tier: 'whale', value: 999, priority: 100, emoji: '🐋' };
  }
  
  // الشريحة 2: تاجر مميز
  if (seller.is_business && (seller.is_verified || seller.has_featured_listings) 
      && seller.total_listings_seen >= 5) {
    return { tier: 'premium_merchant', value: 499, priority: 80, emoji: '🏪' };
  }
  
  // الشريحة 3: تاجر عادي
  if (seller.is_business && seller.total_listings_seen >= 2) {
    return { tier: 'regular_merchant', value: 199, priority: 60, emoji: '🏬' };
  }
  
  // الشريحة 4: بائع موثق
  if (seller.is_verified && seller.total_listings_seen >= 2) {
    return { tier: 'verified_seller', value: 20, priority: 50, emoji: '✅' };
  }
  
  // الشريحة 5: بائع نشط
  if (seller.total_listings_seen >= 2) {
    return { tier: 'active_seller', value: 12, priority: 30, emoji: '👤' };
  }
  
  // الشريحة 6: بائع جديد
  return { tier: 'new_seller', value: 5, priority: 10, emoji: '👤' };
}
```

### أين تُنفّذ دالة التصنيف:

```
1. بعد كل حصادة: classifySeller() لكل seller جديد/محدّث
2. بعد كل enrichment: إعادة تصنيف (ممكن يتغير بعد جلب تفاصيل)
3. Cron يومي: إعادة تصنيف كل المعلنين (لو بياناتهم اتغيرت)
4. عند التسجيل: نسخ seller_tier لـ crm_customers
```

## 💰 كيف نستفيد من التصنيفات لجلب الحد الأقصى من الإيرادات

### استراتيجية 1: التواصل المخصص حسب الشريحة

```
═══ 🐋 حوت — VIP Treatment ═══
الرسالة: شخصية جداً — تذكر اسم المحل + عدد الإعلانات + الفئة
العرض: باقة Diamond مجاناً أول شهر + مدير حساب + نقل إعلانات فوري
المتابعة: كل 24 ساعة (3-5 رسائل) + اتصال هاتفي لو أمكن
الهدف: 50%+ معدل تحويل → 999 ج.م/شهر

═══ 🏪 تاجر مميز — Premium Touch ═══
الرسالة: تذكر نشاطه المميز + عدد إعلاناته
العرض: باقة Gold بخصم 50% أول شهر
المتابعة: 24/48/72 ساعة (3 رسائل)
الهدف: 25%+ معدل تحويل → 499 ج.م/شهر

═══ 🏬 تاجر عادي — Business Focus ═══
الرسالة: قالب تاجر عام
العرض: باقة Silver أول شهر مجاني
المتابعة: 24/48 ساعة (رسالتين)
الهدف: 10%+ معدل تحويل → 199 ج.م/شهر

═══ ✅👤 أفراد — Volume Play ═══
الرسالة: قالب عام بسيط
العرض: "انشر مجاناً"
المتابعة: رسالة واحدة
الهدف: 3-5% معدل تحويل → boost/featured revenue
```

### استراتيجية 2: Upsell حسب الشريحة (بعد التسجيل)

```
مستخدم Free + 10 إعلانات     → عرض Silver (خلص الـ limit)
مستخدم Silver + 50 إعلان      → عرض Gold
مستخدم Gold + مبيعات عالية    → عرض Diamond
أي مستخدم + مزاد ناجح         → عرض Featured للمزاد القادم
```

### استراتيجية 3: تخصيص الموارد

```
80% من وقت التواصل → 🐋 حيتان + 🏪 مميز (10% من المعلنين)
15% من وقت التواصل → 🏬 عادي + ✅ موثق (25%)
5% من وقت التواصل → 👤 أفراد (65%) — AI فقط بدون بشري

لأن:
  حوت واحد = 999 ج.م/شهر
  100 بائع فرد = ~1,200 ج.م/شهر (boost فقط)
  الجهد المبذول: حوت واحد < 100 بائع فرد بكتير!
```
